The universe.sql file defines a complete relational schema for a fictional astronomical universe, including galaxies, stars, planets, and moons. It sets up all tables, relationships, and constraints needed to maintain data integrity, such as primary keys, foreign keys, and unique fields.

The file also includes sample data for each entity, ranging from real astronomical objects to scientifically plausible exoplanets and moons. This provides a ready‑to‑use dataset for exploration, testing, or world‑building within the database.
